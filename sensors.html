Wari LiDAR Distance Sensor

High-precision distance measurement using Garmin Lidarlite v3HP sensor with multiple connectivity options for diverse deployment scenarios.

Key Features

Sensor: Garmin Lidarlite v3HP

Range: Up to 35m (depending on target reflectivity)

Precision: 1 cm

Accuracy: ~5 cm

Solar-powered: 2W solar panel (3G/4G/5G options)

Battery: Lithium-ion 18650 (14500 for 5G option)

Configuration Options

Wari Water Quality Buoy

Self-righting floating buoy for continuous water quality monitoring with turbidity, temperature, and electrical conductivity sensors.

Key Features

  • Turbidity: 0-4000 NTU (Precision: 5 NTU, Accuracy: ±10%)
  • Temperature: -55-315°C (Precision: 0.1°C, Accuracy: ±1%)
  • FEC: 0-2000 µS/cm (Precision: 2 µS/cm, Accuracy: ±10%)
  • Design: Self-righting with closed-cell Neoprene seal
  • Battery: Lithium-ion 15500 battery slot
  • Deployment: Single hitching point

Configuration Options

  • Internal storage only
  • With LoRA radio module

Technical Specifications

Microprocessor Atmega328P (Arduino compatible)
Memory 512 KB EEPROM, 64 MB Flash
Clock Internal high-precision
